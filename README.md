# Automated VM Health Monitoring & Self-Healing System

## Overview
Automated VM Health Monitoring & Self-Healing System is a Python-based infrastructure monitoring solution designed to monitor AWS EC2 instances, detect system anomalies, trigger automated recovery actions, and send proactive notifications to administrators.

## Features
- Real-time VM health monitoring
- CPU, Memory, and Disk utilization tracking
- Windows service availability checks
- Automated service recovery
- Email alert notifications
- Daily monitoring reports
- Centralized logging
- AWS EC2 compatible
- PowerShell integration for self-healing

## Technologies Used
Python, AWS EC2, PowerShell, psutil, boto3, SMTP, Windows Server, CloudWatch
